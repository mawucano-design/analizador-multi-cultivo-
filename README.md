## 🚀 Deploy en Streamlit Cloud

1. Ve a [share.streamlit.io](https://share.streamlit.io)
2. Conecta tu cuenta de GitHub
3. Selecciona este repo
4. Archivo principal: `app.py`
5. **Deploy** → ¡Tu app en vivo!
